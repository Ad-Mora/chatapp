Instructions to run this application:

1. In the terminal, cd into the location of this project (chatapp)
2. Type 'node server' into the command prompt
3. On your web browser, navigate to localhost:8000
4. Enter a username and you will be a the chatlog screen. Type in any message into the prompt.

Note that you can open multiple tabs, and navigate to localhost:8000 on each tab, with a different username each time.
You will be able to see messages from every other "user" in this way, and be able to communicate back and forth.
